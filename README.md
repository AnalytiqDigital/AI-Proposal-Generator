# AnalytIQ ProposalAI

An AI-powered SaaS platform for generating professional business documents including:

- Proposals
- Quotations
- Invoices
- Contracts
- Company Profiles
- Business Plans

Built with:

- React
- Express.js
- MongoDB
- Node.js
- Gemini AI
- JWT Authentication
